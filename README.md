# Deneyap Modüler Donanım Test Konsolu

Deneyap Kart 1A kullanılarak geliştirilen OLED ekranlı ve
joystick kontrollü bir donanım test konsoludur.

## Özellikler

- I2C cihaz tarama
- 9 eksenli IMU kimlik testi
- Canlı ivme, jiroskop ve manyetik alan ölçümü
- GPIO loopback testi
- PWM ile LED parlaklık kontrolü
- Donanım kesmesi ve debounce testi

## Kullanılan Donanımlar

- Deneyap Kart 1A
- OLED ekran
- 9 eksenli ataletsel ölçüm birimi
- Joystick
- LED ve direnç
- Buton
- Breadboard ve jumper kablolar
